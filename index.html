Spotify Statistics App

  • I developed an app that analyzes user listening habits by integrating with the Spotify API. The backend of the app is built using Python and Flask, allowing efficient data processing and smooth API interactions. The app offers users personalized insights into their listening habits, including their top artists, genres, and overall listening patterns. Additionally, it features interactive dashboards that visualize data over time, helping users discover new music based on their preferences. The app was deployed on a cloud server for seamless, cross-device access.

Fashion Trend Analyzer

  • For my Fashion Trend Analyzer project, I analyzed the fashion trends of 2023 using TensorFlow, predicting future trends based on historical data and seasonal patterns. The app scrapes various sources, including fashion blogs, online stores, and social media, to gather trend data. The frontend was developed with React.js and Wix Studio, creating a visually appealing and user-friendly interface. The system uses machine learning models to detect popular colors, materials, and styles, offering predictions about upcoming trends. It also provides regional and demographic insights, helping users understand global fashion shifts.

Recipe Predictor

  • The Recipe Predictor app I created utilizes machine learning algorithms to recommend recipes based on ingredients users already have. By analyzing a large database of recipes, the system matches ingredients and offers meal suggestions, taking into account nutritional preferences and dietary restrictions. The user interface allows for easy input of available ingredients, and the app filters results based on cooking time, difficulty, and cuisine type. Additionally, it integrates real-time ingredient availability from local grocery stores, ensuring users can find convenient and practical meal ideas.
